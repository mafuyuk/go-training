# Golang Training

## 📌 概要
Golangの学習結果保存用リポジトリ

### ディレクトリ構成
```bash
├── design-pattern
│     * デザインパターン学習用
│
├── standard-package
│     * 標準パッケージの学習用
│
├── third-party-package
│     * サードパーティのライブラリの学習用
│ 
└── yukicoder
      * yukicoderのコード格納用
```

## 🌐 動作環境 
* Golang v1.10.0
* Docker Latest Version

## ▶️ 実行方法
### 依存ライブラリのインストール
```bash
$ make setup
```
